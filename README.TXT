PURPOSE OF PROJECT: Create a user-friendly Graphical User Interface (GUI) for efficiently managing various bank cards, including debit and credit cards. The GUI allows users to add new cards, withdraw money, set credit limits, cancel cards, display card information, and maintain an organized list of cards, providing a seamless experience for users without requiring technical expertise or programming knowledge.
